HHKB 中的第一个「H」是什么意思？

这是一个看起来有些无厘头的问题，我曾经也完全没意识到 HHKB 这个名字可能是由四个单词首字母缩写组成的——实际上这四个字母代表的意思是 Happy Hacking Keyboard，所以答案显而易见，HHKB 中的第一个「H」代表着「Happy」。

然而现实是颇多的无辜群众在听闻了「一把就退烧」的豪言壮语之后持币入场，然后带着同样强烈的失落点开闲鱼一键转卖。这简直太不快乐了，并且随着这种 unhappy 的事情越来越多，让 HHKB 看起来似乎变成了一个营销骗局，甚至反向催生了「HHKB 智商税」的风向。

所以想要获得「HHKB 能否给我带来快乐」这个问题的答案，也确实体现着强烈的个人偏好和需求。恰逢少数派商店上架双模板 HYBRID / HYBRID Type-S 两款新品，我们也想跟你聊聊哪些人群适合购买 HHKB，它真的完美适配所有程序员吗？如果你看完后感到心动，不妨点击下方的链接继续了解。

HHKB 是否拥有退烧级别的手感

HHKB 采用静电容轴，按下后，电容的容量会发生改变，实现按键触发效果。由于是无触点设计，所以几乎无磨损。相比机械键盘，无物理触点的设计有着比机械键盘更长的使用寿命，由于静电容是无段落感的，手感方面更接近红轴的体验。

首先我们需要明确的是，HHKB 独特的手感与静电容触发原理并不能完全对应。之所以要强调这件事情是因为看这篇文章的你可能对静电容键盘的触发技术并不感兴趣，但相比之下对 HHKB 与燃风（RealForce）、宁芝（Niz）乃至阿米洛静电容的区别反而比较关注。

虽然四者同为依靠电容量变化来进行开关判断的静电容键盘，但在控制电容量这件事情上，HHKB 和 RealForce 所用的是一款名为 Topre 的胶碗，Niz 用的是自家在 Topre 的专利过期后进行仿制的国产胶碗，而阿米洛那边区别更大，虽然名为静电容，但除了触发原理外轴体的结构更接近机械轴体。

所以提到 HHKB 的手感，我们不能简单地套用「静电容」来进行总结，实际上评价的对象应该是选择了 Topre 胶碗的静电容键盘，因此也正是 Topre 对胶碗的极致调教以及与锥状弹簧间的完美匹配才铸造了 HHKB 的优秀手感。

而关于「退烧」这件事，我的看法是你至少应该发过烧，才能确定自己是不是被治愈了，假使你从未体验过各种类型的机械键盘便从薄膜键盘一步跳到了 HHKB，那严谨点说你是没有办法判断 HHKB 的手感是否就是最适合自己的。

特殊布局：HHKB 与众不同之处

HHKB 取消了数字键、功能键和方向键等非必要按键，只有重要的最少按键。每个按键的键帽大小仍为 19.05 毫米的同时，体积大幅减少。

那么 HHKB 真的完美适配所有程序员的需求吗？这是一个比「HHKB 是否拥有退烧手感」更加复杂的问题。

实际上 HHKB 所谓的键位优化主要针对 UNIX 程序员（Vim/Emacs），通过 Backspace 下移与以及左 Ctrl 变换到 Capslock 的位置，达到以最短的手指移动距离实现全部敲码行为的目的；并且在此基础上进一步调整，原左右 Ctrl 位置留空，右 Shift 裂变出了一颗 1U 的 fn 键。

macOS 用户对于 Ctrl 与 Capslock 的互换不会太陌生，毕竟是系统里面自带的一个偏好设置；而左右下角原 Ctrl 键位的消失，则让 HHKB 的识别性大幅增加，算得上是其布局最大的特色了。

我本人虽然并非程序员，但就日常的实际输入体验下来，左 Ctrl 和 Backspace 的位置变化使常用按键基本集中在 R2-R4 这三排之间，确实分别减弱了左右手的移动负担。但十几二十年的标准 ANIS 布局输入习惯已经养成，初上手 HHKB 前几天的输入效率十分之低；并且最大的问题还是方向键——对于更多的普通用户来说，没有方向键的 60% 布局键盘已经不太容易被接受，而在此基础上继续进行按键调整、删减的 HHKB 布局自然难逃遇冷局面。

所以当除了 UNIX 程序员之外的人群面对着 HHKB 就只能束手无策了吗？

事情当然不是没有别的解决办法。

首先所有涉及到键位布局的问题我们都可以通过学习、适应来解决，就像你最初学习打字那样。我的经验是 Ctrl 和 Capslock 调整后带来的效率提升非常值得我去接受前几天的不习惯，而就码字过程中的方向键需求，Fn+四个按键的组合键也是完全够用且便捷。

我们找了几位使用 HHKB 超过一年的朋友，说说他们的 HHKB 使用体验。在入手之前，先来看看 HHKB 究竟适不适合你？

@涨汤：我是一名互联网的 C 端产品经理，工作里除了组织、沟通以外，主要的输出内容就是以文本为主的需求文档，其实我有 3 把 HHKB，主要使用的型号为 HHKB Pro BT 白色无刻。

工作之余，写作占据了我比较多的时间，例如少数派的《Power+ 2.0》栏目就能看到我写作的内容。目前来说，持续用键盘输入在我的日常里占据了很大的比例，所以希望长时间使用键盘时，手指、手腕都能有比较舒服的体验，且不易疲劳。

老实讲，买第一把 Type-S 的时候，真正让我脑袋发热的还是因为它的外观。 不管是按键的大小、键程、材质还是刻印，每个细节都让人无比舒服。当然，它的内功也十分深厚，用起来绝不掉链子。

关于 HHKB 几年来的体验，其实我 参加征文 的时候已经都写完了，这里就简单总结几点我心中 HHKB 的优点吧：

  • 最喜欢的键位：聊 HHKB 是绕不开键位的，我太喜欢这套布局了，不仅仅是 ⌃Control、 ⎋Esc 等按键不寻常的默认位置，也包括其它按键的间距。
  • 按键十分稳定：持续输入时的表现是我最在意的，这么四五年用下来，按键的表现从来没有让我失望过，始终是相同而且干脆的反馈，从不出错。
  • 没有多余按键：HHKB（US 布局）总共只有 60 个按键，几乎不占桌面空间，但是又有全尺寸键盘的使用面积。
  • 容易保持清洁：同样的清洁习惯，其它键盘明显就更容易变脏；HHKB 的 PBT 材质不仅容易保持清洁，而且还不容易「变旧」，永远是那种经典的感觉。

这些全都是主观感受，所以如果你在犹豫要不要买，建议先找一把试用一段时间。先检查一下钱包，然后跟着你自己的心走。

@眼圈黑：我目前是一名机械类整机制造公司的采购经理，兼职做一部分可配置物料的设计工作，平时除了组织沟通以外就是一些中度的文本输入工作。

如果出差时书包重量允许的话我也会选择带上 HHKB，这样我就可以在出差间隙能以熟悉的方式在更新工作情况，处理文字输入工作。

使用的是 HHKB Pro2（PD-KB400B），之后海淘了一块 hasu 的板子改造成了蓝牙有线双模（不是因为蓝牙款丑，单纯就是因为穷），如果你对 hasu 感兴趣的话可以看一下少数派的 这篇稿子，当然某宝也有了更好的方案。

平时的工作离不开大量的纸质文档，所以桌面对我来说寸土寸金，会选择这类键盘主要原因是因为不占地方，配合蓝牙我能很方便地变换我的桌面布局。在用 HHKB 之前，我买了一款价格相对便宜的 67 键红轴机械键盘，虽然是红轴，还是会担心打扰到别人，最终我选择了 HHKB。

单就我个人的使用经验来看，半天的时间我就爱上了 Control 键位的设计甚至还把我公司的笔记本的键位也做了调整，方便平时出差时保持相对熟悉的键位，保证输入效率。

举个例子：因为我每个月会有一周左右会专心做一些可配置物料的新增或者维护工作，我在处理这项工作时手部动作往往是频繁重复 control + C/V 然后输入一些关键语句，在使用 HHKB 输入时，我的左手基本可以固定在一个位置，输入效率很有保障！

@小波：我是一名产品经理 ，之前从事硬件行业，做物联网的方案。最近一年在一家做中文 NLP 的公司管理项目，主要是负责 toB 业务的交付。工作上和键盘打交道最多应该是写文档，甲方需要的各式各样的文档，正在使用的是 HHKB Pro 2 白色有刻，购于 2014 年。

对一个技术出身，且还是一个 VIM 用户而言，买 HHKB 应该是迟早的事情。

以下是我选 HHKB 的主要原因：

  • 默认的 Control 按键位置，是为了保护我的小手指
  • 原生的 macOS Command 按键位置，因为是 Mac 用户，这个应该是刚需
  • 有限但足够的硬件拨码开关调整键位，不用依赖系统或者 AutoHotKey 之类的软件来修改按键布局，保证了开箱即用
  • 小巧但足够的 60% 键盘布局，这个只是个人喜好

另外就是，HHKB 的「轴」用了一种叫静电容的技术方案。和 Cherry、凯华等使用的传统机械开关式的轴不太一样，当年对这个静电容好奇也是我选择购买 HHKB 的原因之一。

因为我主要的需求是写文档和写代码，从个人使用角度，有几个感受：

  • 60% 键盘布局刚刚好，尤其是 Control 和 Fn 的位置，唯一有点美中不足的应该是「~」的布局，一开始使用可能会非常不习惯，从而会影响 CLI 下快速跳转 Home 目录的情况；Delete 的按键布局位置也刚刚好，不会太远也不会太近
  • 静电容确实在手感和按键反馈上都做得非常不错
  • 基本上满足我「手不离开键盘」的需求
  • 额外的两个 USB 接口，可以插入无线鼠标的接收器，避免浪费现在笔记本电脑上越发紧张的 USB 接口。还可以扩展键盘能力，比如接上一个额外的小键盘

最后

HHKB 依然值得推荐吗？如果你对上面提到的特殊布局不介意，即是一个很少进行竞技类游戏、并且可以忍受 Capslock、F 区和方向键缺失的用户，那这个答案的主观回答毫无疑问是肯定的。当然键盘手感喜好是非常私人的，这并不代表静电容就是什么标准答案，只能说大部分坐在电脑前码字的人应该不会讨厌它。

关于 HHKB 的争议一直存在，作为一把文字输入工具它也承载了太多的象征意义。在薄膜键盘、机械键盘和静电容之间是不应该存在鄙视链关系的，所以思考文章开头的问题：购入一把 HHKB 是否能给你带来真正的快乐，而并非是通过「静电容的」来取悦看客。

版本选择上，毫无疑问更新了双模连接和 Type-C 接口的 HHKB Professional HYBRID 要适合大多数人。Type-S 静音版本加入了静音圈，一定程度上减弱了 HHKB 回弹时的杂音，缺点则是 Type-S 的胶碗段落克数更轻一些，手感也会更「肉」，在部分用户看来显得胶碗有些疲软。

P.S. 关于键帽

另外需要提到的是虽然给机械键盘更换键帽是一件稀松平常的事，然而在 HHKB 上因为火山口造型的「菊花」与常规 MX 轴体键帽并不适配，而 Topre 类静电容键盘的市场又没有大到可以支撑起几家第三方键帽厂，因此在键帽上 HHKB 这类键盘没有太多的个性化选择。除了官方的之外，Topre 自家 RealForce 静电容键盘的拆机键帽也可以转移到 HHKB 上；国产键帽品牌 Domikey 在前几年也针对 HHKB 推出过几款配色还不错的键帽，不过目前似乎也是卖一套少一套的状态。
